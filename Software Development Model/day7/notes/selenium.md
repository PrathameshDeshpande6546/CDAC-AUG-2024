# selenium

## configure browser

```bash

# download the web driver for firefox
# https://github.com/mozilla/geckodriver/releases

# chrome web driver
# https://developer.chrome.com/docs/chromedriver/downloads/version-selection

# extract the zip file and add the executable in OS environment PATH  variable
# to update the path variable
# open .bashrc file
> vim ~/.bashrc

# add the following line
# export PATH=$PATH:<web driver path>

# reload the settings
> source ~/.bashrc

```
